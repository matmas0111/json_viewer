# JSON Tools

[English](README.md)

JSONデータを扱うためのシンプルなWebツール集です。

## ツール一覧

### 1. JSON Viewer（index.html）

[index.htmlを開く](https://matmas0111.github.io/json_viewer/index.html)

JSONデータをツリー構造で表示します。

**機能**

- ツリー形式で表示
- ノードの展開 / 折りたたみ
- 配列の要素数を表示

**使い方**

1. JSONを入力欄に貼り付け
2. 「ツリー表示」ボタンをクリック

---

### 2. JSON Diff Viewer（diff_tool.html）

[diff_tool.htmlを開く](https://matmas0111.github.io/json_viewer/diff_tool.html)

2つのJSONを比較し、差分を表示します。

**機能**

- 2つのJSONの差分を検出
- 変更前 / 変更後の値を表示
- パスを分かりやすく整形して表示

**使い方**

1. 2つの入力欄にJSONを貼り付け
2. 「比較する」ボタンをクリック
