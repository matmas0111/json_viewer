# JSON Tools

[日本語はこちら](README.ja.md)

A simple set of web tools for working with JSON data.

## Tools

### 1. JSON Viewer (index.html)

Displays JSON data in a tree structure.

**Features**

- Tree view representation
- Expand / collapse nodes
- Display array length

**Usage**

1. Paste JSON data into the input field
2. Click the "Build Tree" button

---

### 2. JSON Diff Viewer (diff_tool.html)

Compares two JSON objects and highlights differences.

**Features**

- Detect differences between two JSON inputs
- Show before / after values
- Display differences with readable paths

**Usage**

1. Paste JSON into both input fields
2. Click the "Compare" button
